Role and responsibility:

  • Take ownership of team deliverables and provide estimates for a given task or module.
  • Track and report development progress and performance on a regular basis.
  • Understand client requirements, implement necessary Salesforce Commerce Cloud (SFCC) modules and extensions.
  • Perform necessary configurations like assets, slots, promotion, campaigns, site preferences, and content, using SFCC Business Manager.
  • Design and develop features like customer account, shopping cart, order management, promotions, social commerce, etc.
  • Back-end development using Commerce Cloud script, controllers or pipelines, and Storefront Reference Architecture (SFRA).

Key skills required :

The candidates who are interested to apply in these job openings must have the following key skills that are required to do this:

  • Experience in developing business solutions using SFCC (Demandware).
  • Good knowledge of SFCC or Demandware Script, XML, HTML5, CSS, Ajax, jQuery, JSON object definition, Java OOP or Node.js.
  • Experience in using Pipelines, JS controllers, and real-time data exchange using web services.
  • Hands-on experience of integrating custom services into SFCC. tax, and operation management suite (OMS) integrations.
