Name of the post:

Description:

Informatica has issued a notification to fill the vacancies of various posts of Associate Software Engineer. If you are finding jobs in Bangalore then feel free to apply online from The ProGrad .The candidates who are interested in the vacancies issued by Informatica can see the detailed information about key skills required, educational qualification, total vacancies, how to apply, roles and responsibilities are given below. Informatica is dealing with software products. If you are interested then read the details that are given below.

Total vacancies:

  • Various vacancies have released.

Selection Process:

  • Written test
  • Group Discussion
  • Personal Interview

Role and responsibility:

  • Develop automated tests for product features
  • Ability to root cause defects by analyzing debug logs, analyzing crash dumps and reviewing code
  • Deliver improvements to automation framework, and fill gaps in automation of tests
  • Take the products/module/features through the full software development life cycle
  • Advance best practices such as automated testing, correct prioritization of exploratory tests, thorough SDK documentation, peer reviews, maintaining API compatibility, creating robust, maintainable APIs etc

Application fee:

  • No information is given about the application required.

Educational qualification:

The candidates who are interested to apply for Informatica job must have the following qualification:

  • The candidates must have completed the B.E, B/Tech from a recognized university to apply for the application Informatica QA Engineer 2021

Key skills required :

The candidates who are interested to apply in these job openings must have the following key skills that are required to do this:

  • Self-starter with strong self-management skills
  • Hands-on experience in programming languages like Java
  • Good to have Hands-on experience on Python, PERL or shell scripting
  • Experience in Selenium web driver, Selenium Grid or similar tools.
  • Experience in web services / API testing.

Pay scale info:

  • The Pay scale information will be decided at the time of the interview.

The document needs to be carried:

The candidates who will be selected for the interview must carry the below-mentioned documents:

  • The first and foremost thing is your resume/CV which has your educational background and other related information.
  • You must have your Aadhaar card for identity purposes.
  • You must have other identity proofs like a PAN card, voter id, or Passport to prove your citizenship.
  • At last, you should have photocopies of important educational certificates.

  • You should have an email id and a phone number for communication purposes.
