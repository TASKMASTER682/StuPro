Role and responsibility:

  • Build distributed, highly available and fault tolerant application using Java, - Spring, Kafka and other technologies
  • Build large scale and highly available API's - both internal and external
  • Build, deploy, release, and support application on early stages

Key skills required :

The candidates who are interested to apply in these job openings must have the following key skills that are required to do this:

  • Good hands-on with coding skills
  • Core Java 5+ years
  • Spring framework 5+ years
  • Kafka or other messaging 3+ years
  • Knowledge of key APIs: JPA, JTA, CDI, etc.
  • Knowledge of various design and architectural patterns
  • Understanding of microservices architecture
  • Containerization solutions (e.g. Docker, Kubernetes, Open Shift)
  • Building tools (e.g. Maven, Gradle)
